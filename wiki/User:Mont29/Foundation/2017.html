Week 219 - 11/18 to 11/24

Worked on first review of ID static override, and did some side-work regarding threading in Blender.

  • Misc:
    • Enhanced our bugfix-release log generator script (to hopefully reduce time needed to sort of fixes already cherry-picked in previous release) (rBDT7d02c7).
    • Worked on removing some more OMP code from Blender (rB7770c2ef, rB5f4058dd).
    • Painfully added more atomic primitives to our collection (mainly, signed integer ones, rB105b9583, rB497e2b3d, rB580b34e5).
    • Added non-gcc static assert macros (rBd423e66d), and used it to cleanup our type size checks in atomic_ops (rBff9eab79).
    • Finalized and committed new BLI_task parallel iteration over mempool items (D2930, rBb84e6dfe, rBefb86b71, rBcf6e8edd), used it to add parallel iterators over basic BMesh items (rBbc3f0cfd).
    • Used new BMesh parallel iteration to replace OMP code in normal computing, gives nice 40% speedup as a bonus (rB43ddf0e9).
    • Converted more code from OpenMP to our own BLI_task system.
  • Assets:
    • Updated ID static override patch (D2417).
  • Bug Tracker:
    • BT misc (general check, assign & close work).
    • Fixed #53343 Custom Normal Data Transfer Crashes when some vertexes have no faces (rB78461465).
    • Reviewed D2883 RNA-property tagging using tags defined by struct (+ operator properties tagged 'BASIC').
Fixed: 1; Reviewed:1; Closed: 3;

    • Usual BT & patches work.
