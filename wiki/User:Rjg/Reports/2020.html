Organization

  • Created the blender-triagers channel for discussions between triagers and developers working on the bug tracker
  • Proposal on the handling of complex and older bug reports as reply to Germano Cavalcante's proposal (non-public exchange via email, results and further discussion should be posted by Philipp Oeser on bf-committers in the near future)
  • Created the wiki page "How to write a bug report" that contains a guide for users on how to write good bug reports that would allows to triage them more easily.
  • Created the wiki page "Template texts for bug triaging" with additional template texts that are currently not in the Triaging Playbook.
