Calendar Week 1

Jan 3 - Jan 9

General

  • Moderation and helping people on platforms (devtalk, developer, chat...)
  • Offered the Open Chat again, this week on Tuesday.
  • Triage reports / patches

Module Improvements

  • Further worked on the Module proposal, simplified the status and add additional info. Next steps:
    • Agree on a common (basic) template for the landing pages. I created an example page, which can be used to tinker. Every module can of course add additional information and links, but the basics should ideally look the same for all modules to improve consistency.
    • Announce the implementation of this and give every module a time frame of 1-2 weeks to update their pages. Modules that miss that deadline will be set to inactive.

Release Coordination

  • Postponed Bcon2 until 5th January, due to holidays and some pending features. https://lists.blender.org/pipermail/bf-committers/2022-January/051224.html

Development

  • Fix #94546: Remove soft limit for the Clamp Node value socket (rB070948b5).
